# Release 0.8

## Added

- Simulation Engine
- Simulation Pipeline
- Simulation Context
- Detection Phase
- Simulation Clock
- Structured Simulation Events
- DetectedContact model

### Simulation Engine

- Added mission-based detection probability calculation.
- Introduced `DetectionProbabilityCalculator`.
- Detection logic extracted from `DetectionPhase` into a dedicated calculator component.
- Introduced `SubmarineDetectionModifier`.
- Detection probability is now influenced by the assigned submarine role (SSN/SSBN).
- Simulation probability model now supports chained modifiers.
- Introduced `ContactTypeGenerator`.
- Contact generation now depends on the patrol mission instead of being completely random.
